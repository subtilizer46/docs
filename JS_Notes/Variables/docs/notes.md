## Variable
* It will store the value/data/
* Its like a pendrive/container

## Declaration

* There are 4 ways
* By Using 
    - var
    - let 
    - const keywords
    - Automatically

* Naming Rules
    - Begin with letter,underscore(_),dollar($)

* Case sensitivity
    - myvar and myVar

* Example:
    - var x;
    - let x;
    - const PI;
    - var india;

## Intialization

* var x=10;
* let x=10;

## Reassignment

```
var x=10;
x=20;
```

But we cannot reassigned the variable decalred by const.

const PI=3.14;
